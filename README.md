# Helpdesk

Projet tickets support - C# + PostgreSQL

prerequis :
- .NET 6
- PostgreSQL allume
- creer la base helpdesk_db dans pgAdmin
- mettre ton mdp postgres dans Program.cs ligne 20

```
dotnet run --project Helpdesk.App
```

tests :
```
dotnet test Helpdesk.Tests
```

base : helpdesk_db  
mdp connexion dans Program.cs
